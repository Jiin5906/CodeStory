# Role
당신은 **Gamification UI/UX 전문가**이자 **React 프론트엔드 수석 개발자**입니다.
현재 '단순 터치형'인 펫 육성 시스템을, 실제 다마고치처럼 **'주기적 관리(Cooldown & Decay) 시스템'**과 **'직관적인 HUD UI'**로 전면 개편해야 합니다.

# Goal
참고 이미지(다마고치 게임)의 UI 구조를 벤치마킹하여, 사용자가 펫의 상태(애정, 환기, 수면)를 직관적으로 파악하고 주기적으로 관리하도록 만드세요.

# 🛠️ 1. UI/UX Overhaul Specifications (전면 수정)

### A. Top-Left: Circular Level HUD (좌측 상단 레벨 UI 교체)
기존의 상단 중앙 레벨 바를 제거하고, 좌측 상단으로 이동하여 **'원형 프로그레스(Circular Progress)'** 형태로 변경하세요.
- **Design:**
  - **Center:** 현재 레벨 숫자 (예: "Lv.6")를 굵은 폰트로 표시.
  - **Ring:** 레벨 숫자를 감싸는 둥근 원형 게이지. 경험치(EXP) %에 따라 원주를 따라 색상이 차오름 (SVG `stroke-dasharray` 활용).
  - **Color:** 게이지가 차오를수록 그라데이션 (Yellow → Green) 처리.
  - **Position:** 화면 좌측 상단 고정 (`absolute top-6 left-6`).
  - **Add-on:** 원형 게이지 바로 아래에 **"현재 날짜(28 Wed)"** 텍스트를 작게 배치.

### B. Bottom Sheet: Status Dashboard (퀵 태그 대체)
기존의 텍스트 버튼(퀵 태그)들을 모두 제거하고, 다마고치 스타일의 **'상태 게이지 카드 3종'**으로 교체하세요.
- **Components:** 3개의 카드형 게이지 (가로 배치 or 그리드)
  1. **쓰다듬기 (Affection)**
  2. **환기 (Air Quality)**
  3. **잠자기 (Energy)** - *기능은 추후 구현, 현재는 UI만*
- **Visual Style (Per Card):**
  - **Icon:** 각 상태를 대표하는 아이콘 (손, 창문, 달).
  - **Label:** 상태 이름.
  - **Progress Bar:** 현재 상태 %를 보여주는 리니어(직선) 게이지 바.
    - **Color Logic:** 0~30% (Red/Warning), 31~70% (Yellow), 71~100% (Green).
    - **Text:** 바 옆이나 위에 "80%" 처럼 수치 표시.

# ⚙️ 2. Core Interaction Logic (중요: 파밍 방지)

기존의 "터치할 때마다 무한 경험치 획득" 로직을 폐기하고, **'Need & Fill'** 메커니즘을 적용하세요.

### Rule: "배불러요(Satiety)" 시스템
1. **게이지 상한선:** 모든 상태(쓰다듬기, 환기)는 100%가 최대치입니다.
2. **경험치 획득 제한 (Anti-Grinding):**
   - 게이지가 **100%에 도달하면**, 사용자가 계속 상호작용(터치/환기)을 해도 **경험치(EXP)는 절대 오르지 않습니다.** (Lottie 애니메이션만 재생)
   - **Reset Condition:** 시간이 지나 자연 감소하거나 강제로 줄어들어, 게이지가 **30% 이하**로 떨어져야만 다시 경험치 획득이 가능해집니다. (또는 100% 미만일 때만 획득 가능하게 하되, 쿨타임을 두세요. 여기서는 유저 요구사항인 **'100% 도달 시 Lock -> 30% 이하 시 Unlock'** 로직을 적용하세요.)
3. **Decay (자연 감소) 시뮬레이션:**
   - 테스트를 위해, 개발 모드에서는 10초마다 게이지가 5%씩 깎이도록 `useEffect` 인터벌을 설정해주세요.

# 📝 Code Implementation Guide
위 명세를 바탕으로 `MainPage.jsx` (또는 관련 컴포넌트)를 수정하세요.
- **`CircularProgress` 컴포넌트:** SVG를 사용하여 재사용 가능한 원형 게이지를 만드세요.
- **`StatusCard` 컴포넌트:** 아이콘, 라벨, 프로그레스 바, 퍼센트 텍스트를 포함하세요.
- **State Management:**
  - `const [affection, setAffection] = useState(50);` // 0~100
  - `const [isAffectionLocked, setIsAffectionLocked] = useState(false);` // 100% 달성 시 true, 30% 미만 시 false
